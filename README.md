# cp_MLX90614_display

simple example for Melexis MLX90614 non contact temperature sensor with Adafruit ESP32-S3 Reverse TFT Feather 

## HW

- [uC](https://circuitpython.org/board/adafruit_feather_esp32s3_reverse_tft/)
- [Sensor MLX90614](https://learn.adafruit.com/using-melexis-mlx90614-non-contact-sensors/wiring-and-test)
    - [wiring sensor](https://learn.adafruit.com/using-melexis-mlx90614-non-contact-sensors/wiring-and-test)
    - [color-codes Stemma QT / Qwiic cables](https://learn.adafruit.com/introducing-adafruit-stemma-qt/technical-specs)
    - [Melexis datasheet](https://www.melexis.com/en/documents/documentation/datasheets/datasheet-mlx90614)
